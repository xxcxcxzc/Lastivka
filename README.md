# Ластівка — Месенджер

Повноцінний месенджер з бекендом. Українська мова, біло-блакитна тема, стиль Telegram/Discord.

## Як запустити (фінальний етап)

1. **Встановіть Node.js** (якщо ще немає): https://nodejs.org/

2. **Встановіть залежності:**
   ```bash
   npm install
   cd server
   npm install
   cd ..
   ```

3. **Запустіть бекенд** (перший термінал):
   ```bash
   npm run server
   ```
   Має з’явитися: `Ластівка — бекенд: http://localhost:3001`

4. **Запустіть фронтенд** (другий термінал):
   ```bash
   npm run dev
   ```
   Відкрийте в браузері адресу (наприклад http://localhost:5173).

## Що працює

- **Реєстрація та вхід** — дані на сервері, токен у браузері
- **Канали** — створення та чат у каналах, повідомлення зберігаються на бекенді
- **Особисті повідомлення** — діалоги з іншими користувачами, контакти з імені
- **Пошук** — по каналах і по тексту повідомлень (серверний пошук)
- **Налаштування** — профіль та інформація про конфіденційність

Дані зберігаються на сервері в папці `server/data/` (файли JSON). Усі запити йдуть через API на порт 3001; фронтенд проксує `/api` на бекенд.

## Скрипти

| Команда | Опис |
|--------|------|
| `npm run dev` | Запуск фронту (Vite) |
| `npm run server` | Запуск бекенду (Express) |
| `npm run build` | Збірка фронту для продакшену |

## Запуск як програма (вікно месенджера)

Після встановлення залежностей (`install-Lastivka.ps1` або інсталятор) **ярлик «Ластівка»** відкриває **вікно програми** (Electron), а не браузер. Сервер і інтерфейс працюють на одному порту (3001), дані користувача — у `%APPDATA%\Lastivka\data`.

## Іконка ярлика (кругла)

У папці **logo** є **LastiVka-round.svg** — варіант іконки з круглим тлом. Конвертуйте його в **LastiVka.ico** (онлайн SVG→ICO) і покладіть у **logo\LastiVka.ico**. У `LastivkaInstaller.iss` розкоментуйте `SetupIconFile=logo\LastiVka.ico` і додайте до рядків ярликів: `IconFilename: "{app}\logo\LastiVka.ico"`. Потім перезберіть інсталятор (F9).

## Якщо при кліку на ярлик нічого не відбувається

1. Запустіть **«Ластівка (діагностика)»** з меню Пуск (або файл **Start-Lastivka-debug.bat** у папці програми) — з’явиться вікно з текстом помилки.
2. Переконайтесь, що встановлено **Node.js** (https://nodejs.org) і після встановлення перезапускали комп’ютер або сесію.
3. Одразу після встановлення програми має запуститись **install-Lastivka.ps1** (встановлення залежностей і збірка). Якщо ви його закрили — запустіть вручну з папки програми: права кнопка по `install-Lastivka.ps1` → «Виконати з PowerShell».

## Lastivka.exe (консольний інсталятор)

Щоб зібрати **Lastivka.exe** (інсталятор залежностей у один клік), виконайте в PowerShell: `.\build-LastivkaExe.ps1`

Месенджер називається **Ластівка**.
