# Ластівка — запуск як сервер

Щоб інші ПК могли підключатися до одного сервера (спільні користувачі, канали, повідомлення).

**Варіант А:** Запустіть звичайну Ластівку (`run-Lastivka.ps1`) — сервер уже працює, інші ПК можуть підключатися.  
**Варіант Б:** Запустіть тільки сервер (`run-Lastivka-Server.ps1`) — без вікна програми, лише для мережевого доступу.

## 1. На ПК-сервері

1. **Відкрийте порт у брандмауері** (один раз, від імені адміністратора):
   ```
   ПКМ по open-firewall-Lastivka.ps1 → Запустити від імені адміністратора
   ```

2. **Запустіть сервер**:
   ```
   run-Lastivka-Server.ps1
   ```

3. Запам’ятайте IP-адресу, яку покаже скрипт (наприклад, `192.168.1.100`).

4. На цьому ж ПК можна відкрити браузер: `http://localhost:3001` або `http://192.168.1.100:3001`.

## 2. На інших ПК (клієнти)

1. Відкрийте браузер.
2. Введіть адресу: `http://IP_СЕРВЕРА:3001`  
   Наприклад: `http://192.168.1.100:3001`

3. Зареєструйтеся або увійдіть — усі дані зберігаються на сервері.

## 3. Якщо не підключається

- Переконайтеся, що ПК в одній мережі (той самий Wi‑Fi або LAN).
- Перевірте, що брандмауер дозволяє порт 3001.
- Спробуйте `ping IP_СЕРВЕРА` з клієнтського ПК.
